PURPOSE: 
To implement sorting algorithms using C Language. 
1. Insertion Sort.
2. Counting Sort.
3. Merge Sort.
4. Randomized Quick Sort.
**************************************************************************

COMPILE: 
Created a Makefile that will compile my project. 

Command: $make
gcc -o yash_divecha_proj1 yash_divecha_proj1.c

How to clean object and executable files: 
$make clean
**************************************************************************

EXECUTE: 
Command: $./yash_divecha_proj1
**************************************************************************

IMPLEMENTATION: 
This program asks the user which Sorting algorithm he/she needs to implement. 
1. User will priovide number of elements. (n)
2. n number of elements will be randomly generated. 
3. If(n < 20) then print "*" (stars) for each pass.
